# 🍽️ Hotel Menu Card System (Java)

A simple **Console-Based Hotel Menu Card Application** developed using **Java**. This project allows customers to select food items from Veg and Non-Veg menus, enter quantities, order an additional item, and generate the final bill.

🌐 Live Demo

You can access and test the Hotel Menu Billing System online using the deployed application:

https://untitled-428643488629.asia-southeast1.run.app

## 📌 Features

- Customer name and mobile number input
- Separate Veg and Non-Veg menus
- Menu-driven program using Switch Case
- Select food items and quantity
- Add one more item to the order
- Automatic bill calculation
- Displays customer details and final bill
- User-friendly console interface

## 🛠️ Technologies Used

- Java
- Scanner Class
- Switch Case
- Conditional Statements
- Console Application

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/mehasridhar28/HOTELMENUCARD.git
```

2. Open the project in your preferred Java IDE (Eclipse, IntelliJ IDEA, VS Code, or NetBeans).

3. Compile the program:

```bash
javac HOTELMENUCARD.java
```

4. Run the program:

```bash
java HOTELMENUCARD
```
